# harshit391.github.io

Personal portfolio and web development projects hosted on GitHub Pages.

**Live:** [harshit391.github.io](https://harshit391.github.io)

## Projects

### BharatPe Clone

A pixel-accurate UI clone of the [BharatPe](https://bharatpe.com) fintech platform, built with pure HTML and CSS — no JavaScript.

- Auto-scrolling hero carousel and product tabs powered entirely by CSS `@keyframes`
- Sections: navbar, hero slider, achievements, merchant/consumer offerings, news, footer
- Responsive layout with a mobile breakpoint at 768px

[View Live](https://harshit391.github.io/bharatpeClone/)

### Movie Box (Site1)

A personal portfolio landing page with a clean, modern design.

- Welcome section with call-to-action buttons and decorative geometric shapes
- Navigation bar and testimonials tab structure
- Green accent theme

[View Live](https://harshit391.github.io/site1/)

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Keyframe Animations)
- Google Fonts (Poppins)

## Run Locally

Clone the repo and open any `index.html` in a browser — no build step required.

```bash
git clone https://github.com/harshit391/harshit391.github.io.git
```
